# Hermes Config

Konfigurasi dan skill untuk agen AI **Hermes**.

## Isi

- \`config.yaml\` — konfigurasi utama agen
- \`SOUL.md\` — definisi kepribadian dan panduan agen
- \`memories/\` — memori dan konteks agen
- \`scripts/\` — script pendukung
- \`skills/\` — skill yang dimiliki agen
- \`.env.example\` — contoh konfigurasi environment

## Cara Pakai

1. Salin \`.env.example\` ke \`.env\` dan isi variabel yang diperlukan
2. Gunakan konfigurasi ini bersama platform agen AI yang kompatibel
